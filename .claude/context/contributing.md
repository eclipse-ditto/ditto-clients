# Contributing

## Requirements

- Sign Eclipse Foundation Contributor Agreement
- Sign-off commits with `git commit -s`
- Include tests for non-trivial changes
- Squash commits before PR

## License Headers

EPL-2.0 required on all source files. Use current year for new files.

Java example:
```java
/*
 * Copyright (c) 2026 Contributors to the Eclipse Foundation
 *
 * See the NOTICE file(s) distributed with this work for additional
```
